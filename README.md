# News Capstone Project

This is a Django-based news application built as part of the HyperionDev Capstone Project. The project includes user roles, article management, Sphinx documentation, Git version control, and Docker containerisation.

## Features

- Django web application
- Custom user roles
- Article creation and management
- Role-based dashboards
- MySQL database support
- Sphinx documentation
- Docker configuration
